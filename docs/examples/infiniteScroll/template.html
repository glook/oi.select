Infinite Scroll

Now you can pass oi-options attribute from your controller or pass it as string

If you want to turn on infinite scroll you need to pass $page attribute to your callback function

$scope.oiOptions = 'item.name for item in get($query,$page) track by item.id';
oi-options="oiOptions" throttle="560" threshold="42" ng-model="bundle" placeholder="Select"
